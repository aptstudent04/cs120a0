On Books:

Name Cover Author Summary
Programming Perl Christiansen et al. Perl is like a Swiss Army Knife. There's almost certainly a better tool for what you're trying to do, but if you know Perl you're going to try to do it in Perl before you go digging for something else.
vi and Vim Editors Pocket Guide Robbins, Arnold If you can't open lynx to read the docs on vi because you only have one TTY and you don't want to quit the editor and lose your file chain, it's helpful to have a pocket reference.
AppleScript: The Definitive Guide Neuburg, Matt PowerShell is like AppleScript for Windows. It doesn't do a lot on its own, but you can use it to tell various other system components to do useful things, which can save time from repetitive tasks like telling that one person how to do the thing, again. Just script it. Everyone wins!
sed & awk Dougherty, Dale and Robbins, Arnold sed and awk are two very powerful tools for text editing, but beware! Extended use of the stream editor may lead to frustration when using graphical editors, and Outlook doesn't support vim keybindings!
